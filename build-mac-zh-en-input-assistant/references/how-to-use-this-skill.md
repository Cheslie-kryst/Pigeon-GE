# 如何把这个 Skill 交给 AI 使用

这个仓库不是一个已经打包好的 App，而是一套让编程 AI 在你的 Mac 上自动搭建、安装并检查“中译英输入助手”的完整说明和模板。

## 准备

- 一台 Mac，建议 macOS 13 或更高版本。
- 建议至少 8 GB 内存，并预留约 5 GB 磁盘空间。
- 一个能够读取文件、编写代码和执行终端命令的编程 AI。
- 安装和下载过程中需要联网；日常使用默认可以本地翻译。

## 使用步骤

1. 在 GitHub 页面选择 **Code → Download ZIP**。
2. 解压整个文件夹，不要只复制 `SKILL.md`，因为 AI 还需要里面的脚本、源码模板和参考资料。
3. 把解压后的文件夹交给你的编程 AI，或让 AI 打开该文件夹。
4. 把下面这段话发给 AI：

> 请完整读取这个文件夹里的 SKILL.md，并使用 $build-mac-zh-en-input-assistant 在我的 Mac 上搭建并安装中译英输入助手。优先使用本地 Ollama，不启用 DeepSeek；所有下载、系统权限、启动程序和安装操作都先征得我同意。请完成验证后再告诉我成功。

5. AI 会先检查电脑环境，再说明需要下载什么。确认后，它会创建程序、准备本地模型、构建、安装和检查。
6. macOS 弹出辅助功能提示时，进入 **系统设置 → 隐私与安全性 → 辅助功能**，打开“中译英输入助手”，然后退出并重新打开程序。

## 安装完成后怎么使用

1. 在微信、备忘录、TextEdit 或其他文本框中输入中文。
2. 先确认中文已经真正输入到文本框，不要停留在拼音候选状态。
3. 按 `Control + Shift + Space`。
4. 稍等片刻，中文会被替换成自然的美式口语英语。

程序会出现在 Mac 顶部菜单栏。可以在那里切换翻译风格、查看当前翻译方式或退出程序。

## 隐私说明

- 默认使用本机 Ollama，输入文字不会发送到云端。
- 程序不会记录键盘输入、翻译历史或使用统计。
- 只有你主动配置并选择 DeepSeek 时，待翻译文字才会发给 DeepSeek。
- DeepSeek 密钥存放在 macOS 钥匙串中，不应写进代码或上传到 GitHub。
- 某些软件需要用复制粘贴兼容模式，因此系统剪贴板内容可能被临时替换。

## 常见情况

- 第一次本地翻译可能需要 20–30 秒，模型启动后通常会更快。
- 如果快捷键没有反应，确认程序仍在菜单栏运行，并检查是否有其他软件占用了同一快捷键。
- 如果一直提示辅助功能权限，让 AI 检查实际启动的 App 路径和签名，不要反复开关权限。
- 密码框不会被读取或翻译，这是安全设计。
- 目前只支持 Mac，不包含 iPhone、iPad、Windows 或 Android 版本。
