# 中译英输入助手使用说明（Mac）

## 它能做什么

在普通文本框中输入并确认中文后，按 `Control + Shift + Space`，程序会把中文替换为自然、地道的美式英语。程序常驻 Mac 顶部菜单栏。

## 第一次使用

1. 确认 Ollama 已运行，并已安装 `qwen3:4b-instruct` 模型。
2. 打开“中译英输入助手”。
3. 进入 **系统设置 → 隐私与安全性 → 辅助功能**。
4. 打开“中译英输入助手”的权限。
5. 退出程序，再重新打开一次。

只需要授予“辅助功能”。不需要完全磁盘访问、录屏、麦克风、通讯录或定位权限。

## 日常使用

1. 在微信、备忘录、TextEdit 等软件的文本框中输入中文。
2. 先让中文真正出现在文本框中，不要停留在拼音候选状态。
3. 按住 `Control` 和 `Shift`，再按一下空格键。
4. 等待翻译完成，原中文会被英语替换。

第一次调用本地模型可能需要约 20–30 秒；模型已经启动后通常会更快。

## 翻译方式

- **本地 Ollama（默认）**：文字只在你的 Mac 内处理，隐私最好。
- **DeepSeek（可选）**：可能更快，但待翻译文字会发送到云端。只有你主动配置并选择它时才会使用。

DeepSeek 密钥应保存到 macOS 钥匙串，不要放进文档、源码或聊天截图。

## 隐私和限制

- 程序不会记录你的键盘输入、翻译历史或使用统计。
- 密码框和安全输入框不会被读取或替换。
- 某些软件不完整支持 macOS 辅助功能接口，程序会在你按快捷键后使用复制粘贴兼容方式；这可能临时改变剪贴板内容。
- 翻译过程中如果焦点切换到其他软件，程序不应把结果粘贴到错误位置，而会保留到剪贴板供你手动粘贴。
- 当前版本只支持 Mac。

## 遇到问题

- **快捷键打开了系统搜索**：请确认使用的是 `Control + Shift + Space`，不是 `Option/Alt + Space`。
- **没有找到文本框**：先确认中文已经输入完成，光标仍在可编辑文本框；可先在 TextEdit 中测试。
- **反复要求辅助功能权限**：退出程序，确认权限授予的是目前安装位置的 App，再重新打开。仍然反复时，让维护者检查 App 签名，不要持续开关权限。
- **本地模型无法连接**：打开 Ollama App，确认菜单栏中有 Ollama 图标，然后重试。
- **翻译出现分析过程**：确认安装并选择的是 `qwen3:4b-instruct`，不是 `qwen3:4b`。

## 卸载

退出程序后，把“中译英输入助手.app”移到废纸篓即可。Ollama 和模型是独立软件，不会随本程序自动删除；如需删除，请先确认你没有其他程序在使用它们。
