# Runway Video Script Lab

本地运行的视频脚本拆解与 Runway 提示词反推工具。上传视频后，它会在本机后端抽取音频和画面帧，并调用用户自己的 OpenAI API Key 做：

- 原视频脚本拆解
- 原视频 Runway 反推提示词
- 新产品脚本仿写
- 仿写脚本 Runway 提示词
- 中文 / English 切换
- 局部手动修改与 AI 二次修改

## 隐私与 Key

默认不需要把 API Key 写进代码或仓库。用户在网页里输入自己的 OpenAI API Key，Key 只会发送给本机后端用于当次请求，不写入文件，也不放进浏览器本地存储。刷新页面后需要重新输入。

仓库只保留 `.env.example`，不要提交 `.env`。

## 本地运行

需要：

- Node.js 20+
- ffmpeg
- OpenAI API Key

启动：

```bash
npm start
```

不要直接复制示例网址。必须先在自己电脑运行 npm start，再打开终端显示的本地地址。

如果端口被占用：

```bash
PORT=4183 npm start
```

## 工作流

1. 在网页输入 OpenAI API Key
2. 上传参考视频
3. 可选：补充字幕 / 口播文本
4. 选择产品品类和目标风格
5. 打开「需要脚本仿写」后，上传新产品图、填写新产品名称
6. 可选填写「仿写要求」
7. 点击「生成结果」

生成后：

- 可以直接编辑任意中文或英文板块
- 中文修改后可点击「同步英文」
- 手动修改仿写脚本后，点击「根据脚本更新提示词」
- 想让 AI 局部改仿写脚本，填写「仿写二次调整」并点击「AI 修改仿写」

## Codex Skill

仓库包含一个可复制安装的 Skill：

```text
.codex/skills/runway-video-script-lab/
```

把该文件夹复制到你的 Codex skills 目录后，即可用 `$runway-video-script-lab` 让 Codex 按这个工作流协助搭建、运行或改造工具。

## 不建议直接部署公网

这个项目是本地工具。虽然 Key 不保存，但浏览器会把 Key 发给本机后端。如果要部署成多人在线服务，需要重新设计认证、密钥代理、配额、安全审计和文件清理策略。
